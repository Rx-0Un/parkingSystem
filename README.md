# parkingSystem
本系统是毕业设计的一部分，分为android端和后台两个部分，此为web端的代码
本项目是一个简单的停车收费系统，主要采用的技术有Spring Boot+MyBatis+Thymeleaf框架,bootstrap作为前端框架(模板网站https://www.matrixadmin.wrappixel.com/)
